# vDTC International — Dashboard

**vDTC International** is a research and development corporation building next-generation medical, infrastructure, and emergency response technologies.

---

## What Is This Dashboard?

This GitBook is the central knowledge base for the vDTC portfolio — covering active projects, architecture specifications, investment details, regulatory pathways, and design team collaboration. It is the single source of truth for engineering, clinical, and business teams working across all vDTC initiatives.
