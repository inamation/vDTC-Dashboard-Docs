# HVF Phase 2 Controls

**High-Voltage Controls Platform — Phase 2 Upgrade**

HVF Phase 2 Controls is the second-generation control systems upgrade for the vDTC high-voltage fabrication platform — improving safety interlocks, real-time monitoring, and AI-driven fault prediction for high-voltage manufacturing and test environments.

---

## Core Scope

- **Safety interlock system** — redundant hardware interlocks for HV isolation; IEC 61508 SIL 2 compliant
- **Real-time monitoring** — voltage, current, temperature, and arc detection across all HV circuits
- **AI fault prediction** — LSTM time-series model trained on HV event logs; predictive maintenance alerts
- **SCADA integration** — Modbus/OPC-UA interface to facility SCADA; remote monitoring dashboard
- **Emergency disconnect** — automatic HV isolation on fault detection; <5ms response time
