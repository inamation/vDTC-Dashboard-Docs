# MDN Initiatives

**Multi-Domain Network Infrastructure Initiatives**

MDN Initiatives encompasses vDTC's multi-domain network infrastructure development — the connectivity fabric that links emergency response, medical, industrial, and infrastructure systems into a unified, resilient network.

---

## Core Scope

- **Private LTE/5G** — CBRS-band private cellular deployments for 911 field operations and industrial sites
- **Mesh networking** — LoRaWAN and 802.11s mesh for disaster-degraded environments where cellular fails
- **Network slicing** — QoS-prioritized traffic lanes for 911Hub (emergency) vs. analytics vs. management
- **Zero-trust security** — NIST SP 800-207 compliant; mTLS everywhere; hardware attestation on edge nodes
- **Connectivity resilience** — multi-path: fiber + CBRS + Starlink + LoRa fallback; automatic failover

---

## Integration Points

- 911Hub PSAP connectivity (primary and failover paths)
- JOBSITE911 and industrial site private LAN
- 911AirDrop deployed shelter site comms
- Luminary data center uplink diversity
- IronShield NERC CIP-compliant network isolation
