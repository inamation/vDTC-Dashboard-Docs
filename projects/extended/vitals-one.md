# VitalsOne

**"Always-On Awareness. When Seconds Matter Most."**

VitalsOne™ is a real-time biometric wristband integrated with the 911 Emergency Medical Informatics ecosystem. Designed for children, adults, elderly, and high-risk individuals, it provides continuous vitals monitoring with automatic escalation to 911Hub when thresholds are crossed.

---

## Core Technology

- **Biometric sensors** — heart rate, SpO2, respiratory rate, skin temperature, galvanic skin response
- **BLE 5.2 + LTE-M** — dual connectivity; local relay via phone or direct cellular for independent operation
- **911Hub integration** — MQTT vitals stream; threshold breach triggers PSAP alert with GPS location
- **AI anomaly detection** — time-series pattern recognition for cardiac events, falls, and respiratory compromise
- **FHIR R4** — vitals data formatted as Observation resources for hospital handoff

---

## Target Users

- Pediatric monitoring (school, home, transit)
- Elderly independent living
- Post-discharge cardiac and respiratory patients
- First responders and field personnel
