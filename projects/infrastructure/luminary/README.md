# Luminary Architecture — Data Center

**Hyperscale AI data center campus powered by IronShield SMR — hospital-grade uptime, zero-carbon compute.**

Luminary is a purpose-built hyperscale data center campus at the Fort Custer site, powered by the IronShield Small Modular Reactor. Designed from ground up for AI inference workloads — high GPU/TPU density, liquid cooling, extreme uptime — Luminary provides the compute substrate for all vDTC AI products and serves external enterprise and government tenants.

---

## Design Specifications

| Parameter | Value |
|---|---|
| Total IT load capacity | 40 MW (Phase 1) → 200 MW (Phase 3) |
| Power source | IronShield SMR (100% carbon-free) |
| Cooling | Rear-door liquid cooling + free-air economization |
| PUE target | ≤1.08 (vs industry average 1.5) |
| Uptime class | Tier IV equivalent (99.9999%) |
| Connectivity | 100G dark fiber ring — Chicago / Detroit / Cleveland |
| Certifications target | ISO 27001, SOC 2 Type II, HIPAA-compliant colocation |
